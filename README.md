## Quick Start

1. **Clone the repository:**
```bash
git clone https://github.com/Pratikkk0566/AI_Agent_Clone_Trial.git
cd AI_Agent_Clone_Trial
```

2. **Install dependencies:**
```bash
npm install
```

3. **Set up your API key:**
Create a `.env` file and add your Groq API key:
```bash
GROQ_API_KEY=your_groq_api_key_here
PORT=3000
```

4. **Start the server:**
```bash
npm start
```

5. **Open your browser:**
Navigate to `http://localhost:3000`
